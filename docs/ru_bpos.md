# Как настроить свой БПОС-сервер для доступа к отечественным сервисам из-за границы?
## БПОС - что такое?
**БПОС - безопасное подключение к отечественным сервисам**. Для чего это вообще нужно? Пример: вы решили отдохнуть заграницей (Китай) летом. Вы самозанятый. Вдруг, настало 9 число месяца (день оплаты налогов самозанятым), но вы не можете оплатить налог, т.к. не имеете доступа к ЕГР ФНС. В этом случае пригодится доступ к своему собственному БПОС.
## Подготовка
Будем считать, что у Вас дома есть home lab (дословно домашняя лаборатория/сервер). Тогда подготовка к конфигурации БПОС займет немного времени. Поставим на home lab `Ubuntu Server LTS` (LTS - long term support).
![Фоточка](./img/ubuntu.png)
## Выбор софта для развертывания
Наиболее популярные решения для развёртывания собственного БПОС:
- OpenVPN — классический, надёжный, гибкий.
- WireGuard — современный, простой в конфигурации, высокопроизводительный.
- SoftEther VPN — многофункциональный, поддерживает множество протоколов.

Основываясь на наших потребностях, мы выбрали `WireGuard`.
## Установка WireGuard и ufw
Для начала обновим все пакеты:
```bash
sudo apt upgrade
```
Далее установим wireguard и ufw
```bash
sudo apt install wireguard ufw
```
- ufw - удобное управление файрволлом системы (менеджер файрволла)
