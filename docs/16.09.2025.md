# Системы счисления. Особенности округления чисел

### 1. Единицы измерения количества информации
- **Бит (bit)** – минимальная единица информации (0 или 1).  
- **Байт (Byte)** = 8 бит.  
- Кратные единицы: Кбайт, Мбайт, Гбайт и т.д.  
- Логарифмическая мера: количество информации зависит от числа возможных состояний.

### 2. Позиционные системы счисления
- Характеризуются **основанием (q)**.  
- Число записывается как сумма разрядов, умноженных на степени основания.  
- Примеры:  
  - **Двоичная (q=2)**: используется в ЭВМ.  
  - **Восьмеричная (q=8)** и **шестнадцатеричная (q=16)** – удобны для представления двоичных чисел.  
  - **Десятичная (q=10)** – привычная для человека.  

### 3. Оптимальная система счисления
- Теоретически оптимальной является система с основанием **e ≈ 2,718**.  
- На практике удобнее использовать целые основания (2, 8, 10, 16).  

### 4. Нетрадиционные системы счисления
- **Система с основанием -2** (негативные основания).  
- **Фибоначчиева система счисления**.  
- Используются редко, в спец. задачах.  

### 5. Округление чисел
- **В большую сторону** (ceil).  
- **В меньшую сторону** (floor).  
- **До ближайшего целого** (round).  
- Возможна потеря точности.  

### 6. Непозиционные системы счисления
- Значение цифры не зависит от позиции.  
- Пример: **римские числа (XIV = 14)**.  
- Ограниченное применение, т.к. неудобно для вычислений.
