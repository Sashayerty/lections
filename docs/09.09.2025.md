>Статистический подход реализуется в теории информации, когда кол-во информации определяется уменьшением состояние системы.

### Свойства системы:
1. Целостность и делимость
2. Наличие устойчивых связей между элементами
3. Организация
4. Этмерженность

>Семантический подход: для измерения смыслового содержания информации используется тезаусная система Шнейдера.

Кол-во информации нелинейно зависит от состояния индивидуального тезауса пользователя.

>Прагматический подход: количество информации рассматривается с точки зрения достижения получателем поставленной цели

Количество информации о некотором объекте определяется непредсказуемостью состояния, в котором находится объект.
### Свойства меры, кол-ва информации:
1. Неотрицательность
2. Принцип предопределенности (если об объекте все известно, то количество информации о нем равно 0)
3. Агитимность (кол-во информации о системе в целом будет представлять собой сумму информации о каждом элементе по отдельности)
4. Монотонность (кол-во информации при монотонном изменении объекта информация о нем тоже меняется монотонно)
### Есть сигналы двух типов:
- Аналоговые (раньше; прим.: голос)
- Цифровые (сейчас)
